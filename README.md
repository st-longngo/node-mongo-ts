# Requirement

- Install [Node.js](https://nodejs.org/en/) version 20.0.0

# Getting started

- Clone the respository

```
git clone  <git lab template url> <project_name>
```

- Install dependencies

```
cd <project_name>
npm install
```

- Copy file env

```
cp .env.example .env
```

- Build and run the project

```
npm run dev
```

Navigate to `http://localhost:8080`
